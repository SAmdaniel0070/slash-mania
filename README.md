## 🍉 GAME OVERVIEW

> Your mouse is your **blade**.  
> Fruits fly. You slice.  
> One bomb… and it's over.

**Slash Mania** is a fast-paced arcade game inspired by *Fruit Ninja*, built using **Python + Pygame**, focusing on gameplay feel, combo mechanics, and clean UI.

---

## 🎮 CORE FEATURES

- 🔪 **Knife Cursor Gameplay**
- 🔥 **Dynamic Combo Multiplier**
- 💣 **Bomb = Instant Game Over**
- 🎵 **Immersive Sound System**
- 🔇 **Mute / Unmute Toggle**
- 🎨 **Retro Pixel Aesthetic UI**
- ⚡ **Optimized Smooth Gameplay**

## 🎯 GAME RULES
- 🍎 Slice fruits → gain points
- 🔥 Combo increases score
- ❌ Miss = combo reset
- 💣 Bomb = GAME OVER
- 🎯 Bomb appears every 3–4 waves

## 🧮 SCORING SYSTEM
- Action	Result
- Slice fruit	+Combo multiplier
- Chain slices	Higher score
- Miss fruit	Combo reset
- Hit bomb	☠ Game Over
